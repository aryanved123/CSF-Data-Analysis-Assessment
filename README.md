# CSF Data Analyst Internship Assessment

## Project Overview
This notebook explores the relationship between cheese production and provincial weather across Canada. Using data from the Canadian Cheese Directory and average temperatures by province, the project identifies patterns and trends through visualizations and analysis.

## Files
- `cheese_weather_analysis.ipynb`: The main notebook with all code, visualizations, and analysis.
- `data/cheese_directory.csv`: Dataset of Canadian cheese entries.
- `data/canada_weather.csv`: Dataset with average temperatures per province.
- (Optional) `data/canada_temperature.csv`: Contains population and other demographics.
